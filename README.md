#  Web Security Scanner

**Web Security Scanner** is a full-stack web application designed to analyze the security posture of websites. The application performs automated checks on SSL/TLS usage, HTTP security headers, and cookie configurations, then generates an interactive analysis dashboard and downloadable PDF reports.


##  Features

*   **Automated Security Scanning:** Checks SSL/TLS certificates, HTTP security headers (HSTS, CSP, X-Frame-Options, etc.), and cookie flags (Secure, HttpOnly).
*   **Intuitive Dashboard:** Displays clean visualizations of detected vulnerabilities, risk levels, and an overall security score.
*   **PDF Reporting:** Generate and download comprehensive security audit reports to archive or share results.
*   **Secure Authentication:** User registration and login workflows fully protected via JWT (JSON Web Tokens).


## 🛠️ Tech Stack

Frontend:Angular
Backend: Node.js, Express 
Database: MongoDB 
Authentication: JWT (JSON Web Tokens) 
Reporting PDFKit / Puppeteer
